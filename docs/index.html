System Architecture

The agent interacts with the game engine via a custom Python Gym environment. It uses the TMInterface plugin to inject inputs directly into Trackmania Nations Forever over a TCP socket. The Proximal Policy Optimization (PPO) neural network takes the current car state (velocity, angle, and distance to target) as input, calculates the optimal action based on previous training, and outputs continuous steering, gas, and brake values to drive the car.

Training & Strategy Insights

We successfully implemented multi-instancing, which effectively doubled our training speed and the total number of episodes we could process. Over the course of the project, the agent completed approximately 40,000 episodes.

While the model achieved convergence, it ultimately developed an inferior driving strategy. Instead of braking optimally, the agent learned that wall-tapping to reduce speed was the most effective way to slow down and make tight corners. We explored multiple paths for adjusting the reward structure and map interpretation to discourage this behavior, which are fully detailed in our system breakdown video.

Quick Start

Get up and running with the training environment in minutes.

01

Setup Game & Interface

Install Trackmania Nations Forever from Steam, and inject the TMInterface plugin.

02

Clone & Install

bash 
git clone https://github.com/Flynn-Worner/Trackmania_deepLearn.git
cd Trackmania_deepLearn
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
03

Train the Model

Ensure the game is open and running TMInterface on port 8483.

bash 
python main.py --ports 8483 --timesteps 5000000
